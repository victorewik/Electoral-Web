# 🏛️ Vote for the Public University - UCM Faculty of Law

Official campaign landing page for the student coalition **Derecho x La Pública** for the Faculty Board and University Cloister elections on April 15th.

This platform is designed to mobilize students in defense of public education, exposing the €33.1M adjustment plan affecting the Complutense University of Madrid (UCM) and presenting the coordinated coalition's candidates.

## Key Features

- **High-Impact Visual Identity:** Aesthetic inspired by physical campaign posters, using bold typography (Anton & Archivo Black) and the corporate "Referendum Green" (`#00bf63`).
- **Mobile-First Responsive Design:** Specifically optimized for mobile devices with a fixed horizontal header layout to maintain visual impact on small screens.
- **Secure Suggestion Box:** A direct communication channel with the assembly that stores inquiries in a plain text file (`.txt`) for privacy and ease of management.
- **Strategic Information:** Sections dedicated to the "Why Vote for Us?" manifesto, covering the Master’s access fees, scholarships, and financial transparency.
- **Candidate Directories:** Complete list of representatives for the student board and the university cloister.

## Technology Stack

- **Backend:** Python 3.11 with Flask.
- **Frontend:** HTML5, CSS3, Tailwind CSS.
- **Typography:** Anton and Archivo Black (via Google Fonts).
- **Deployment:** Docker & Docker Compose.
- **Reverse Proxy:** Ready for Caddy integration.

## Installation & Deployment

### Prerequisites
- Docker and Docker Compose installed on your server.
- An existing Docker network (e.g., `porlapublica`) if deploying behind a reverse proxy.

### Setup Instructions
-This program is created to be deployed in a DOCKER-COMPOSE
